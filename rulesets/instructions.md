# Story instructions

Add global rules that apply to all stories. Edit this file to change defaults quickly.

Guidelines
- Make it simple language. the kids are 4-7 years old.
- Try to build the story with a happy ending. With a moral good point that is good for kids. Friendship, trust, adventure.


Examples:
- It's Halloween themed
- Santa is always the bad guy
- All characters are animals
- Stories take place in a magical forest
